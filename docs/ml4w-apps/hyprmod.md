# HyprMod

> Please note that the ML4W Hyprland Settings app has been sunsetted has been replaced with HyprMod

This applications supports you to customize many details of your Hyprland installation. You can overwrite the existing configuration with custom values without adding complex configuration files.**

## Installation

HyprMod is not included in ML4W OS directly and needs to be insatlled separately.

You can install HyprMod directly from the Welcome App and Settings menu or as described on the HyprMod GitHub Page.
https://github.com/BlueManCZ/hyprmod

As soon as HyprMod is installed and after the restort of ML4W OS, a launch button for HyprMod will appear in the Welcome App and Sidebar App.

You can also start the HyprMod from the terminal with 

```sh
hyprmod
```
