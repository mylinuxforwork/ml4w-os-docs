# Monitor Setup

If you want to change the default monitor configuration, you can use nwg-displays.

Open nwg-displays from the application launcher or from the welcome app.

![image](/monitor1.png)

NWG-Displays will write the configuration to 

`./config/hypr/monitors.lua`


You can also create your own monitor variation in ~/.config/hypr/conf/monitors and select it from the Dotfiles Settings app.
